# Lofter-Saver
本软件是为了解决lofter批量下载问题而设计的<br>
使用纯python编写<br>
## 软件特性
1.反编译了lofter客户端，直接使用api获取内容，速度极快<br>
2.使用pywebio实现webui，具有良好跨平台性<br>
3.网页本地化保存支持良好，离线保存网页与原网页基本毫无区别<br>
4.支持批量下载，可解析作者、tag、标签等<br>
5.可导入css样式表自定义webui样式<br>
6.（未实现）下载完成可转换成电子书格式<br>
